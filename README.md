A simple desktop POS/billing app for a Bangladeshi-style Kacchi restaurant, built with Python Tkinter and SQLite. It lets staff quickly select menu items, generate bills, apply vouchers and record order history – all in one window.

Main features

Login system for Employees + Admin

Menu loaded from SQLite DB with admin editor (add / edit / reorder items)

Multi-category menu (Kacchi Combo, Drinks & Dessert, Add-ons, Sharing Platter) with qty buttons and live item totals

Order Summary page with scrollable item list, voucher code apply, VAT, discount and final bill calculation

Multiple payment methods: Cash, bKash, Nagad, Rocket, Card (with cash change/due calculator)

Voucher management (create codes, max uses, usage tracking, soft delete) stored in vouchers.json

Order history viewer with per-bill item details, stored in orders.json

Employee management (add/remove employees, passwords) stored in SQLite

Simple, print-friendly UI themed as “Kacchi Bhai Style Restaurant Billing – Bangladesh”
